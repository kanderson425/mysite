    Hillbillies vs. Aliens

    Hillbillies vs. Aliens is a fun 2D shooter game that forces you to defend your backyard from the likes of Alien scum. Inspired by Duck Hunt, this game combines HTML Canvas with basic JavaScript game logic to provide a fun twist on the classic game. Play it!

    National Park Tracker App

    Inspired by my love of the outdoors, this application utilizes node.js, ejs, Google OAuth and the National Park Service API to provide an application used to track and learn more about the National Parks of the U.S.A!

    GuitarCollector

    Built using Python and Django, the GuitarCollector application combines two of my favorite passions: guitars and organization. I use this application to log which guitars have been played and how many hours I have practiced on each one. With Django in the backend, I can sign-up or login to access my saved collection.
